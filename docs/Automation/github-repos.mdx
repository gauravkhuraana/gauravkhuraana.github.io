# GitHub Repositories

Explore open-source automation testing frameworks and resources to accelerate your test automation journey. These repositories include production-ready frameworks with AI integration and curated testing practice sites.

## Automation Frameworks with Agentic AI

### Playwright TypeScript with Agentic AI
Modern end-to-end testing framework using Playwright and TypeScript, enhanced with AI-powered test generation and maintenance capabilities.

**Repository**: [PlaywrightTypeScriptWithAgenticAI](https://github.com/gauravkhuraana/PlaywrightTypeScriptWithAgenticAI)

**Key Features**:
- TypeScript-based Playwright framework
- AI-assisted test creation and debugging
- Page Object Model implementation
- Cross-browser testing support

---

### REST Assured Framework with Agentic AI
Comprehensive API testing framework built with REST Assured and Java, featuring intelligent test generation through AI integration.

**Repository**: [RestAssuredFrameworkWithAgenticAI](https://github.com/gauravkhuraana/RestAssuredFrameworkWithAgenticAI)

**Key Features**:
- REST Assured for API testing
- AI-powered test generation
- Request/Response validation
- Comprehensive reporting

---

### Selenium Framework with Agentic AI
Robust web automation framework using Selenium WebDriver with Java, integrated with AI capabilities for smarter test automation.

**Repository**: [SeleniumFrameworkWithAgenticAI](https://github.com/gauravkhuraana/SeleniumFrameworkWithAgenticAI)

**Key Features**:
- Selenium WebDriver implementation
- AI-enhanced test maintenance

- Cross-browser compatibility
